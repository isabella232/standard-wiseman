# Wiseman For Standard

A child theme designed by 8BIT for Standard Theme 2 and updated by Jason Bradley for Standard Theme 3.

## Links

* [Standard Theme 2 Customization Download](http://support.8bit.io/entries/373942-customization-the-wiseman-standard-theme-1-0-2-0)
* [8BIT](http://8bit.io)
* [Jason Bradley](http://everchangingmedia.com/)

### Features

* Includes style.less file
* Custom wiseman repeatable background image
* Blue content background color
* White link colors, underline on hover
* Added bottom border for widget list items
* White widget link colors, underline on hover
* Custom post format quote and link background colors
* Custom Activity widget tab and link colors
* Custom footer area and subfloor transparent background color
